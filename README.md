TinosaurDemo

Small demo project focusing on ECS architecture in C and gun implementation.

**🎮 [Play on itch.io](https://narawitp.itch.io/tinosaurdemo)**

---

## Technical Specs
* **Language:** C
* **Architecture:** Entity Component System (ECS)
* **Core Feature:** cam shake (trauma/decay model)
* **Build:** Optimized for low overhead
